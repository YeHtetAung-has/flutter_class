**FLUTTER COMMANDS**

- how to create flutter project
  # flutter create project_name
  
- how to check flutter version
  # flutter --version
  # flutter doctor
  # flutter doctor --verbose (details)

- how to run flutter project_name
  # flutter run 
  # flutter run -v (details)
  
- how to build android and ios for application
  # flutter build apk --release
  # flutter build ios
